Dans certains tests psychotechnique pour les recrutements dans le milieu de l'aéronautique, il y a une épreuve 
d'association qui permet d'évaluer les capacités des candidats à associer une forme a une couleur, en un laps de temps relativement
restreint, le tout durant une période de temps qui peût être relativement longue, afin de tester l'attention et la capacité de
concentration du candidat dans le temps.

Ceci est mon premier programme VisualBasic, je l'ai développé sous le framework .net avec l'IDE VisualStudio.
Au delà du fait qu'il m'a donné un premier apercu de la programmation en VB, il peut également servir de base si un jour vous comptez
passer ce type de sélections.


Règles du Jeu :

Durant 6 minutes vous aurez des formes qui aparaîtront.
- Si un rond apparaît, vous devez appuyer sur la touche gauche de votre clavier sauf si c'est un rond rouge auquel cas vous ne devez appuyer sur aucune touche
- Si un carré apparaît, vous devez appuyer sur la touche bas de votre clavier sauf si c'est un carré vert auquel cas vous ne devez appuyer sur aucune touche
- Si un losange apparaît, vous devez appuyer sur la touche droitede votre clavier sauf si c'est un losange bleu auquel cas vous ne devez appuyer sur aucune touche
Une session dure 6 minutes mais lors de vos sélections cela pourra aller jusque 18 minutes, vous pouvez donc faire 3 sessions d'affilées.
Durant une minute 30 si vous vous trompez une boite de dialogue vous dira que vous vous etes trompé et vous rappellera les règles.
Souvenez vous bien ces commandes car au dela de ce temps vous n'aurez plus d'aide!
Bon entrainement / Bon jeu !

In some psychotechnics tests for aeronautic recruitment, there may have a psychotechnic association test which allow to the company
to evaluate the ability of the candidate to associate a form with a colour, in a relatively short amount of time.
The test may last a period of time long enough to evaluate the attention and the ability of the candidate to stay focused.

This is my first programm in VisualBasic, I developped it under the framework .net, with the IDE Visual Studio.
Beyond the fact that it gave me a first experience with VB' programmation, it also may help you if you are to go through these kind of tests


Rules of the game :

During 6 minutes, some forms will appear.
- If a circle appear, push left, except if it's a red circle, then you don't have to push any keys
- If a square appear, push down, except if it's a green square, then you don't have to push any keys
- If a diamond appear, push right, except if it's a blue diamond, then you don't have to push any keys
One session last 6 minutes, but during the selections i may last until up to 18 minutes, so I advise you to play 3 session in a row to get closest as real conditions.
During 1 minute 30 if you get wrong, a user prompt will appear telling you that youre wrong, and will remind you the rules.
Try to Remember the rules because beyond this time there will be no more help!
Have a nice Training / Gaming !
